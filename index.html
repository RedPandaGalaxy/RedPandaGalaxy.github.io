I am a Gameplay Programmer that specializes in Unreal Engine!
My favourite thing is to implement mechanics and see how they come to live in a project! Creating fun experiences is what drives me.

Projects

Amnesty of the Exiled

Unreal Engine 4.26

UI Programmer, Project Manager

Amnesty of the Exiled is a narrative-driven first-person mystery exploration game in which the player needs to judge the fallen deities of an island and uncover the secrets they hide.
My role was that of a Project Manager and UI Programmer. I designed and implemented Journal System in Blueprints using widgets. It dynamically populates to journal based on the collected items.

This game has been created during a course of the Master Study in Games at ITU in Copenhagen

My responsibilities:

  • UI Programming - made a journal system
  • Implement and design the collectables system
  • Do time management and planning
  • Ensure clear communication and transparency
Unreal Engine 4.26
Team Size: 6 people
Duration: 3 months

Unreal Ryth Lizard

Unreal Engine 5.1

Unreal Rhythm Lizard is a 3rd-person action-platformer game with a non- human character controller in which the playable character is a mecha-lizard - a lizard with robotic parts

My responsibilities:

  • Implement character controller for non-human character
  • Implement and design character abilities - shield, light & heavy attack, grapple hook
  • Adjust scene lightning settings
Unreal Engine 4.26
Team Size: 6 people
Duration: 3 months
